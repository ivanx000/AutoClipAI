# Video Files

Place your viral TikTok-style clips here:

- `clip1.mp4` - Left panel
- `clip2.mp4` - Center panel  
- `clip3.mp4` - Right panel

## Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Aspect Ratio**: 9:16 (vertical/portrait)
- **Resolution**: 1080x1920 recommended
- **Duration**: 10-30 seconds (they loop)
- **File Size**: Keep under 10MB each for fast loading

## Optional Poster Images

Add static poster images for loading state:
- `poster1.jpg`
- `poster2.jpg`
- `poster3.jpg`

## Tips

1. Use high-energy, visually engaging clips
2. Bright colors work well with the frosted overlay
3. Content with movement catches attention
4. Avoid clips with lots of text (they'll be blurred)
